# 중고나라 크롤링 소스코드

## 사용법
1. 해당 링크[https://chromedriver.chromium.org/downloads]를 참고하여 WebDriver 를 설치하고 라이브러리를 설치한다.
2. module/defind.py 에 설치한 드라이버 경로(절대 경로) 를 포함하여 검색할 물건, 네이버 아이디, 비밀번호 등을 설정한다.
3. module/main.py 를 실행한다.
